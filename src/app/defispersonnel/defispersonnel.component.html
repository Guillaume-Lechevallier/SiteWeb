À propos des défis

Ici, je met les défis que je me pose sur ce site, par exemple, le formulaire de contact, que j'ai réalisé, c'est le moyen pour moi de vous montrer ce que je peux faire et de me poser quelques défis


Ce système de mail passe par un application personnalisé de Google.


Après avoir récupéré les identifiants d'application personnalisé, nous pouvons utiliser les principales fonctionnalitée, mais via nos applications.


Tout ceci est hébergé sur l'hébergeur de code python en ligne : pythonanywhere.com


Afin que notre application Angular (Ce site) et notre application Python communiquent entre eux, nous passons par une API, ici, j'utilise Flask (framework pour Python) qui permet à notre application Angular d'envoyer des requettes HTTP à notre application en python


Cela permet une meilleur sécuritée, par exemple, si le code source de notre application Web fuite et est accessible de tout le monde, les identifiants d'applications Google ne sont pas divulgé.
