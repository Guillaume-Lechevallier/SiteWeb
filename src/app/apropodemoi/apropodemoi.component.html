À propos de ce site

Ce site est développé avec le Framework Angular, et tous les appels d'API sont faites vers un script Python, utilisant Flask, un Framework très léger qui me permet de faire des appels d'API. Le site internet est stocké sur Firebase et le script Python est hébergé chez moi.
